# Gentle Care Nursing Website Rebuild

## Business
Gentle Care Nursing

## Primary Goal
Build a modern, highly trustworthy, highly converting website and digital enquiry system that helps families, NDIS support coordinators, hospital discharge planners, and referral partners easily understand services, trust the brand, and submit enquiries or book consultations.

## Core Outcomes
- Increase qualified enquiries
- Improve trust and professionalism
- Make referral capture frictionless
- Improve local SEO and Google visibility
- Integrate website forms and booking logic with GoHighLevel CRM
- Enable AI chat + AI voice website assistant via GoHighLevel widget/embed
- Create a premium digital experience that positions Gentle Care Nursing as a leading provider in its field

## Target Audiences
- Families looking for care services
- NDIS support coordinators
- Hospital discharge planners
- Healthcare professionals making referrals

## Core Services
- NDIS services
- Community nursing services / DVA clients
- Aged care / Support at Home services
- Private nursing and personal care

## Key Messages
- Registered NDIS provider
- Registered DVA community care provider
- In-home nursing and care services
- Personalised, high-quality care
- Smaller client load for more dedicated support
- Easy and fast enquiry process
- Strong trust, professionalism, and responsiveness

## Positioning Goal
The website should position Gentle Care Nursing as a premium, trustworthy, highly responsive care provider that makes it easy for families and referrers to get support quickly.

## Website Strategy
This is not a brochure website.
This is a high-converting healthcare website built to:
- build trust fast
- reduce friction
- improve enquiry conversion
- support SEO growth
- drive qualified referrals
- support backend automation and follow-up

## Proposed Pages
- Home
- About
- Services
- NDIS Services
- DVA / Community Nursing
- Aged Care / Support at Home
- Private Nursing & Personal Care
- Referral / Request Care
- Contact
- FAQ
- Privacy Policy

## Integrations
- GoHighLevel CRM (sub-account)
- LeadConnector mobile app
- Domain / DNS
- Google Business Profile
- Google Calendar if used for bookings
- Form automations via GoHighLevel
- AI chat widget embed code
- AI voice website assistant embed code
- Analytics / Search Console if available

## Design & CRO Standards
The website must be:
- premium and visually polished
- highly converting
- SEO-aware from the ground up
- mobile-first and fully responsive
- calm, trustworthy, and healthcare-appropriate
- easy to navigate for older users and busy coordinators
- built using strong CRO, UI, UX, and information hierarchy principles

## Influence / Inspiration
Apply the strategic principles and influence of:
- Caleb Ulku (SEO and search intent)
- Arsh Sanwarwala / ThrillX (data-driven website structure, conversion-focused design)
- Alex Hormozi (clear offer, value communication, friction reduction)
- Sabri Suby (direct response, conversion architecture, lead generation)

Use these influences to create a website that is structurally stronger, more persuasive, and more effective than typical healthcare provider websites.
