# Non-Negotiables

- This site must feel premium and category-leading
- Mobile UX must be excellent
- Every page must have clear conversion paths
- Trust and clarity come before visual flair
- The design must be calm, structured, and modern
- No bloated code or unnecessary dependencies
- Forms must be easy to complete
- The site must be SEO-aware from the ground up
- The final result must outperform typical healthcare provider websites in clarity, trust, and enquiry flow
